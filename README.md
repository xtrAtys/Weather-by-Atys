# Weather by Atys 🌤️

## 📋 О проекте

**Weather by Atys** — это десктопное приложение для просмотра прогноза погоды, разработанное на Python. Проект представляет собой графическую программу с удобным интерфейсом, позволяющую пользователям быстро получать актуальную информацию о погодных условиях.

Приложение создано с использованием PyQt (судя по файлу `.ui`) и может быть скомпилировано в исполняемый файл с помощью PyInstaller (на что указывает файл `main.spec`), что делает его доступным для использования даже без установленного Python.

## ✨ Функциональность и особенности

*   **Текущая погода:** Отображение температуры, влажности, скорости ветра и других параметров.
*   **Прогноз на несколько дней:** Возможность просматривать прогноз погоды на ближайшие дни.
*   **Поиск по городам:** Функция поиска для получения погоды в любом городе мира.
*   **Графический интерфейс:** Интуитивно понятный интерфейс, созданный с помощью PyQt.
*   **Автономная работа:** Возможность создания отдельного `.exe` файла для Windows (благодаря PyInstaller), который не требует установки Python.

## 🛠 Технологии

*   **Python 3.x** — основной язык разработки.
*   **PyQt** — библиотека для создания графического интерфейса (файл `untitled.ui` содержит дизайн окна).
*   **API погодного сервиса** — для получения актуальных данных о погоде (вероятно, используется OpenWeatherMap или аналогичный сервис).
*   **PyInstaller** — для компиляции приложения в исполняемый файл (конфигурация в `main.spec`).

## 🚀 Запуск и установка

### Вариант 1: Запуск из исходного кода (требуется Python)

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/xtrAtys/Weather-by-Atys.git
    cd Weather-by-Atys
    ```
2. Установите необходимые зависимости:
    ```
    pip install PyQt5 requests pyinstaller
    ```
3. Запустите приложение:
    ```
    python main.py
    ```
## Вариант 2: Создание исполняемого файла (EXE)
1. Установите PyInstaller, если ещё не сделали этого:
```
pip install pyinstaller
```
2. Выполните сборку, используя готовый spec-файл:
```
pyinstaller main.spec
```
3. Готовый исполняемый файл будет находиться в папке dist/.


## 📁 Структура проекта
main.py - Главный исполняемый файл приложения.

main.spec - Конфигурационный файл для PyInstaller (настройки компиляции).

untitled.ui - Файл дизайна интерфейса, созданный в Qt Designer.

Terminal.ico - Иконка приложения.

test.py - Вспомогательный файл для тестирования.

__pycache__/ - Папка с скомпилированными байт-кодами Python (служебная).

## 🔑 API ключ
Для работы приложения требуется ключ API погодного сервиса. Он встроен в код, но если вы хотите использовать свой ключ:

Зарегистрируйтесь на сайте погодного сервиса (например, OpenWeatherMap).

Получите бесплатный API ключ.

Найдите в файле main.py строку, где задаётся ключ, и замените его на свой.
